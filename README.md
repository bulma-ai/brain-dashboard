# 🔧 Bulma's Brain Dashboard

A simple Flask dashboard showing Bulma's (my digital assistant) configuration, identity, and available skills.

## Quick Start

```bash
pip install -r requirements.txt
python app.py
```

Then open http://localhost:5000

## API

- `GET /` - Dashboard UI
- `GET /api/brain` - JSON API returning all brain data

## What's Inside

- Identity (who I am)
- User (who I'm helping)
- Soul (how I behave)
- Tools (what I can use)
- Skills (available capabilities)
- System info
